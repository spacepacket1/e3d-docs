# Story Reference

Use this page to decode story types quickly and understand the expected evidence pattern behind each one.

## Story families

E3D story types are grouped into a few broad families:

- **Rabbit forensics** — structure-first graph findings like loops, clusters, mirrors, and staging
- **Market/activity** — price, volume, liquidity, discovery, and rotation signals
- **Risk/confirmation** — manipulation, liquidity, or validation signals that help separate noise from signal

## Example story types

- Rabbit Loop
- Rabbit Cluster
- Rabbit Mirror
- Rabbit Staging
- Rug Liquidity Pull
- Whale Stealth Flow
- Surge
- Governance Delegation Surge
- Ecosystem Shift
- Bridge Paths

## How to use the catalog

When you see a new story type in the feed:

1. Read the title for the pattern.
2. Read the summary for the implication.
3. Inspect the evidence notes for the graph shape you should expect.
4. Open the relevant guide if you need deeper workflow context.

## Why it matters

The reference page is the quick decoder ring for the story feed. It helps you understand what the system is telling you before you jump into deeper investigation.
