# Python SDK

## Install

```bash
pip install e3d-sdk
```

Requires Python 3.8+. No external dependencies — stdlib only.

## Quick start

```py
import os
from e3d import E3DClient

client = E3DClient(api_key=os.environ['E3D_API_KEY'])
openapi = client.get('/openapi')
rate = client.get('/rate')
print(openapi)
print(rate)
```

## Token profile

```py
from e3d import E3DClient

client = E3DClient(api_key=os.environ['E3D_API_KEY'])
profile = client.get('/token/0x6488861b401F427D13B6619C77C297366bCf6386', query={'chain': 'ETH'})
print(profile)
```

## Stories

```py
from e3d import E3DClient

client = E3DClient(api_key=os.environ['E3D_API_KEY'])
stories = client.get('/stories', query={'limit': 5})
addresses = client.get('/stories/addresses')
print(stories)
print(addresses)
```
