# E3D Docs

Public developer documentation site for E3D, built with Nextra 3 on Next.js 15.

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run start
```

## Contents

- API reference powered by the live `https://e3d.ai/openapi` schema
- Getting started and authentication guides
- Concept pages for Stories, Theses, Token Intelligence, Price Screener, NFTs, and Agents
- JavaScript and Python SDK docs
- Practical guides for token profiles, story feeds, counterparty risk, swap integration, embeds, and hedge fund workflows

## Notes

- Docs live on port 3006.
- Auth management stays on `e3d.ai`; this site only links to signup and key management.
- Swap docs describe wallet-side helper flows only.
